<div id="actionDesc">
<p>
Changes a GameObject's position over time to a supplied destination.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
TODO</div>
</div>

<div id="paramRow">
	<div id="paramName">
Id</div>
	<div id="paramDesc">
iTween ID. If set you can use iTween Stop action to stop it by its id.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Transform Position</div>
	<div id="paramDesc">
Move To a transform position.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Vector Position</div>
	<div id="paramDesc">
Position the GameObject will animate to. If Transform Position is defined this is used as a local offset.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Time</div>
	<div id="paramDesc">
The time in seconds the animation will take to complete.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Delay</div>
	<div id="paramDesc">
The time in seconds the animation will wait before beginning.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Speed</div>
	<div id="paramDesc">
Can be used instead of time to allow animation based on speed. When you define speed the time variable is ignored.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Space</div>
	<div id="paramDesc">
Whether to animate in local or world space.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Ease Type</div>
	<div id="paramDesc">
The shape of the easing curve applied to the animation.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Loop Type</div>
	<div id="paramDesc">
The type of loop to apply once the animation has completed.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Orient To Path</div>
	<div id="paramDesc">
Whether or not the GameObject will orient to its direction of travel. False by default.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Look At Object</div>
	<div id="paramDesc">
A target object the GameObject will look at.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Look At Vector</div>
	<div id="paramDesc">
A target position the GameObject will look at.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Look Time</div>
	<div id="paramDesc">
The time in seconds the object will take to look at either the Look Target or Orient To Path. 0 by default</div>
</div>

<div id="paramRow">
	<div id="paramName">
Axis</div>
	<div id="paramDesc">
Restricts rotation to the supplied axis only.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Move To Path</div>
	<div id="paramDesc">
Whether to automatically generate a curve from the GameObject's current position to the beginning of the path. True by default.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Look Ahead</div>
	<div id="paramDesc">
How much of a percentage (from 0 to 1) to look ahead on a path to influence how strict Orient To Path is and how much the object will anticipate each curve.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Transforms</div>
	<div id="paramDesc">
A list of objects to draw a Catmull-Rom spline through for a curved animation path.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Vectors</div>
	<div id="paramDesc">
A list of positions to draw a Catmull-Rom through for a curved animation path. If Transform is defined, this value is added as a local offset.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Reverse</div>
	<div id="paramDesc">
Reverse the path so object moves from End to Start node.</div>
</div>

</div>
