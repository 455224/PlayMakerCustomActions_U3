<div id="actionDesc">
<p>
Get the next connected player properties. 
Each time this action is called it gets the next child of a GameObject.This lets you quickly loop through all the connected player to perform actions on them.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Loop Event</div>
	<div id="paramDesc">
Event to send for looping.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Finished Event</div>
	<div id="paramDesc">
Event to send when there are no more children.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Index</div>
	<div id="paramDesc">
The player connection index.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Ip Address</div>
	<div id="paramDesc">
Get the IP address of this player.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Port</div>
	<div id="paramDesc">
Get the port of this player.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Guid</div>
	<div id="paramDesc">

</div>
