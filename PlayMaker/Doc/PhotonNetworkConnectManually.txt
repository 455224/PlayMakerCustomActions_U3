<div id="actionDesc">
<p>
Connect to the photon server by address, port, appID and game(client) version.This method is used by ConnectUsingSettings which applies values from the settings file.)</p>
</div>
<div id="paramRow">
	<div id="paramName">
Server Address</div>
	<div id="paramDesc">
The master server's address (either your own or Photon Cloud address).</div>
</div>

<div id="paramRow">
	<div id="paramName">
Port</div>
	<div id="paramDesc">
The master server's port to connect to.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Application ID</div>
	<div id="paramDesc">
Your application ID (Photon Cloud provides you with a GUID for your game).</div>
</div>

<div id="paramRow">
	<div id="paramName">
Client Game Version</div>
	<div id="paramDesc">
This client's version number. Users are separated from each other by gameversion (which allows you to make breaking changes).</div>
</div>
