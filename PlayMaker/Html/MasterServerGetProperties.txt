</div>
<div id="paramRow">
	<div id="paramName">
Ip Address</div>

The IP address of the master server.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Port</div>

The connection port of the master server.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Update Rate</div>

<div id="paramRow">
	<div id="paramName">
Dedicated Server</div>

Flag to report if this machine is a dedicated server.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Is Dedicated Server Event</div>
	<div id="paramDesc">

<div id="paramRow">
	<div id="paramName">
Is Not Dedicated Server Event</div>
	<div id="paramDesc">
